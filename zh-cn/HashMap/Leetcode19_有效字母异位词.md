## 🧠 解题思路
1.  什么是anagram: 相同的字母，换个顺序，就是 anagram
2.  char 类型表面上是字符，但在计算机底层，它实际上是一个整数, 'a' 对应的数值是 97
3.  我们需要把 'a' ~ 'z' (97 ~ 122) 这段数字，平移到 0 ~ 25 这个范围，才能放进数组里。方法就是减去基准值，也就是减去 'a' (97)
4.  先判断长度，如果长度不一样，肯定不是anagram
5.  我们要证明两个字符串里的字符数量完全相等。通常有两种方法  1.用两个数组（比较麻烦）
    统计 s 中 'a' 有 3 个。
    统计 t 中 'a' 有 3 个。
    比较 3 == 3？相等。
    对每个字母都这样比一遍。
    方法二：用一个数组
    我们要证明 s 的数量等于 t 的数量，也就是证明：
    s 的数量 - t 的数量 = 0
    我们把 count 数组看作是一个计数器，初始全是 0。
    s 负责“存入”：遇到一个字符，就 ++（加1）。
    t 负责“取出”：遇到一个字符，就 --（减1）。
    如果 t 真的是 s 的异位词，那么 s 存进去多少，t 就应该正好能取出多少。
    最后，数组里应该干干净净，全部变回 0。
6. 老是忘记数组怎么定义 int[] array = new int[n]; 左方右new.  n为数组大小

- **空间复杂度**: O(1)
